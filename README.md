# Cinema Rental System

Система управления прокатом фильмов для кинотеатров.

## Описание

Desktop-приложение на базе Avalonia UI для управления:
- **Кинотеатрами** - добавление, редактирование и удаление информации о кинотеатрах
- **Каталогом фильмов** - управление базой фильмов с информацией о жанрах, годе выпуска и стоимости
- **Поставщиками** - база данных поставщиков фильмов
- **Арендой фильмов** - учет аренды фильмов кинотеатрами
- **Валидация данных** - проверка корректности вводимых данных
- **Хранение данных** - сохранение в JSON формате

## Технологии

- **.NET 8.0** - платформа разработки
- **Avalonia UI** - кроссплатформенный UI фреймворк
- **MVVM** - архитектурный паттерн
- **JSON** - формат хранения данных

## Запуск проекта

```bash
cd CinemaRentalSystem
dotnet restore
dotnet run
```

## Структура проекта

```
CinemaRentalSystem/
├── Models/          # Модели данных (Cinema, Film, Rental, Supplier)
├── ViewModels/      # View Models для MVVM
├── Views/           # XAML представления
└── Assets/          # Ресурсы приложения
```
