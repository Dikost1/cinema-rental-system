Руководство пользователя (CinemaRentalSystem)
============================================

Описание разработанного приложения (руководство пользователя)
------------------------------------------------------------

Данное приложение предназначено для ведения учета кинопроката (аренды кинолент) между поставщиками и кинотеатрами. Пользователю предоставляется возможность добавлять и редактировать справочники кинотеатров, поставщиков и фильмов, а также оформлять аренду фильмов на заданный период с указанием сумм оплаты и пени. Данные сохраняются в файл формата JSON и автоматически загружаются при следующем запуске приложения.

При запуске приложения открывается главное окно с навигацией по разделам: «Кинотеатры», «Поставщики», «Фильмы», «Аренда». (рисунок 1)

В разделе «Кинотеатры» выполняются основные проверки:
- обязательность названия;

- сумма пени не может быть отрицательной.


Сохранение и загрузка данных
----------------------------

Сохранение и загрузка выполняются автоматически через файл `cinema_rental_data.json`.
При запуске приложения данные загружаются из файла (если файл существует). При добавлении/обновлении/удалении кинотеатров, поставщиков, фильмов и аренд данные сохраняются автоматически.

Файл данных располагается рядом с исполняемым файлом приложения. При необходимости начать работу “с чистого листа” можно закрыть приложение и удалить файл `cinema_rental_data.json`, после чего при следующем запуске будет создан новый пустой файл.
