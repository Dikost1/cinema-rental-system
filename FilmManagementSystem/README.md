# Система учета кинофильмов

## Описание

Консольное приложение на C# для учета информации о закупке кинофильмов. Приложение позволяет хранить информацию о фильмах и поставщиках, с автоматическим сохранением и загрузкой данных из текстовых файлов.

## Функциональность

### Учет поставщиков
- Название поставщика
- Юридический адрес
- Банк и номер счета
- ИНН поставщика

### Учет фильмов
- Название кинофильма
- Категория фильма (боевик, триллер, комедия и др.)
- Автор сценария
- Режиссер-постановщик
- Компания-производитель
- Год выхода на экран
- Связь с поставщиком
- Стоимость приобретения

### Возможности
- ✅ Добавление, просмотр и удаление поставщиков
- ✅ Добавление, просмотр и удаление фильмов
- ✅ Просмотр фильмов по конкретному поставщику
- ✅ Автоматическое сохранение данных при выходе
- ✅ Загрузка данных при запуске
- ✅ Защита от удаления поставщика, у которого есть фильмы

## Структура файлов

- `Program.cs` - основная программа с меню
- `Film.cs` - класс для представления фильма
- `Supplier.cs` - класс для представления поставщика
- `DataManager.cs` - класс для управления данными и работы с файлами
- `suppliers.txt` - текстовый файл с данными поставщиков
- `films.txt` - текстовый файл с данными фильмов

## Формат хранения данных

### suppliers.txt
Каждая строка содержит данные одного поставщика, разделенные символом `|`:
```
ID|Название|Юридический адрес|Банк|Номер счета|ИНН
```

### films.txt
Каждая строка содержит данные одного фильма, разделенные символом `|`:
```
ID|Название|Категория|Сценарист|Режиссер|Производитель|Год|ID_Поставщика|Стоимость
```

## Как запустить

### Вариант 1: Через Visual Studio
1. Откройте файл `FilmManagementSystem.csproj` в Visual Studio
2. Нажмите F5 для запуска

### Вариант 2: Через командную строку
```bash
cd FilmManagementSystem
dotnet run
```

### Вариант 3: Компиляция и запуск
```bash
cd FilmManagementSystem
dotnet build
cd bin/Debug/net6.0
./FilmManagementSystem
```

## Требования

- .NET 6.0 или выше
- Операционная система: Windows, macOS или Linux

## Пример использования

1. Запустите программу
2. Выберите "1. Управление поставщиками"
3. Добавьте поставщика с необходимой информацией
4. Вернитесь в главное меню
5. Выберите "2. Управление фильмами"
6. Добавьте фильм, указав ID существующего поставщика
7. При выходе данные автоматически сохранятся
8. При следующем запуске все данные будут загружены автоматически

## Особенности реализации

- Данные сохраняются в простых текстовых файлах с разделителем `|`
- ID для новых записей генерируются автоматически
- Невозможно удалить поставщика, у которого есть фильмы
- Валидация данных при вводе (год выпуска, стоимость и т.д.)
- Кодировка UTF-8 для поддержки русского языка
- Автоматическое сохранение при выходе из программы
